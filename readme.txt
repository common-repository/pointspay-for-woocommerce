Accept payment via Pointspay in your WooCommerce store.

== Description ==
Pointspay is an innovative loyalty marketing and payment solution, available to leading online stores worldwide. With Pointspay, customers can spend and collect points or miles from their favorite loyalty programs.

With the Pointspay plugin, you can easily install Pointspay on your WooCommerce online store.

= Usage =
Just create an account with us on [our site](https://www.pointspay.com/online-shops-retailers), install the plugin on your WooCommerce store, configure your API credentials, enable it, and you are ready to start accepting payments with us!

Once your customers choose to pay using Pointspay, they will be redirected to our  payment page where they will complete the payment and get back to your store.

= Features =
Generate extra revenue, grow your visibility, and attract a wealth of brand new customers who are incentivised to shop with you.

== Installation ==
1. You can:
 * Upload the `pointspay-wc-gateway` folder to `/wp-content/plugins/` directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **Pointspay for WooCommerce** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Set up plugin on *WooCommerce -> Settings -> Payments -> Pointspay*.
4. For obtaining credentials, please register at [Pointspay](https://www.pointspay.com/online-shops-retailers)
